搭博客

2020-09-16

今天用hexo框架搭了一个博客

配置环境

  • node.js

  • git

  • hexo
    我用的都是最新版本的

    配置SSH key

  • 使用GitBash检查存在的SSH密钥

  • 然后再输入下边的命令后，按回车键3次

  • 然后把SSH密钥粘贴在在github的个人设置中

    检测是否成功

  • 接着配置

写博客

现在我就是在用Markdown语法来写博客

  • 先安装hexo框架

  • 在电脑上新建一个文件夹来存放自己的代码

  • 生成和启动服务

选择主题

  • 在官方的主题中选择一下博客的主题，我选的是比较简洁的一个模板

  • 同时还要修改_config.yml中的theme:landspace``为theme:cards```，然后重新生成

    上传到github

  • 继续修改_config.yml

然后在gitbash中输入hexo d将代码全部提交
